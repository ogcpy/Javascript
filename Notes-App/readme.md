# Notes App

## Description
A web app to create and manage notes.

## Features
- Add, edit, and delete notes.
- Persistent storage using local storage.

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
