# QR Code Generator

## Description
A web app to generate QR codes for URLs or text.

## Features
- Generate QR codes instantly.
- Download QR code as an image.

## Technologies Used
- HTML
- CSS
- JavaScript
- QR Code Library

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
