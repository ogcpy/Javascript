# To-Do List with Webpack

## Description
A to-do list app built with Webpack.

## Features
- Add, edit, and delete tasks.
- Built with Webpack.

## Technologies Used
- HTML
- CSS
- JavaScript
- Webpack

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.
