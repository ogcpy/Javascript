# Tic Tac Toe

## Description
A classic Tic Tac Toe game built with JavaScript. Play against a friend or against the computer.

## Features
- Two-player mode.
- Responsive design.
- Win/draw detection.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
