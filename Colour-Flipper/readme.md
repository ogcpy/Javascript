# Colour Flipper

## Description
A fun app that randomly changes the background color of the page.

## Features
- Random color generation.
- Simple and clean UI.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
