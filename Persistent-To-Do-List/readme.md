# Persistent To-Do List

## Description
A to-do list app with persistent storage.

## Features
- Add, edit, and delete tasks.
- Tasks saved in local storage.

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
