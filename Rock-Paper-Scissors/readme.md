# Rock Paper Scissors

## Description
A simple Rock Paper Scissors game built with JavaScript.

## Features
- Play against the computer.
- Track wins, losses, and ties.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
