# Palindrome Checker

## Description
A simple app to check if a word or phrase is a palindrome.

## Features
- Case-insensitive checking.
- Instant results.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
